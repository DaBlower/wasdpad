# obob's wasdpad
This is my submission to hackpad (v5?)
It has 6 switches, rgb, an OLED display and 3 different layers, WASD mode, arrow key mode and media (changing layers and volume eg)

## Features
* 8 Neopixels
* A 0.91 inch OLED screen
* 6 Cherry MX Switches
* KMK firmware with layers and support for neopixels and the oled

* 6 Keycaps

* 4 M3x16mm screws
* 4 M3x5mx4mm heatset inserts
* 1 3D printed case (find the CAD files in /case)
